# 📌 Major Projects

## BenchmarkTrader (Full-Stack Trading Platform)

BenchmarkTrader is a comprehensive trading platform developed using the **TERN stack (TypeScript, Express.js, React, Node.js)** with **Prisma ORM** and **Supabase** integration.

The platform was designed to simulate an intermediary investment system where clients connect with brokers to manage and execute investments.

### Key Features
- Broker-client matching system
- Investment workflow automation
- Secure user authentication
- Portfolio and trading management
- Scalable backend architecture

*Note: This project is currently in a private repository.*

## WildGraph (Graph-Based Conservation Mapping System)

WildGraph is a **Java-based application** that applies **graph theory and pathfinding algorithms** to environmental conservation challenges.

The system analyzes terrain data and identifies optimal conservation corridors to assist in wildlife protection planning.

### Key Features
- Graph data structures for terrain modeling
- Pathfinding algorithms for route analysis
- Algorithmic optimization for environmental mapping
- Collaborative system design and implementation

*Note: This project is currently in a private repository.*
